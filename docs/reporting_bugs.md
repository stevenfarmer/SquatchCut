# Reporting Bugs

If you run into a problem with SquatchCut (crash, bad layout, overlaps, or confusing behavior), please report it so it can be fixed.

## Where to report

- GitHub Issues: https://github.com/stevenfarmer/SquatchCut/issues/new

## What to include

When you open a bug report, please include:

- What you were trying to do (e.g., "Import CSV and run nesting on a 4x8 sheet").
- Steps to reproduce:
  1. Open FreeCAD
  2. Open SquatchCut
  3. Import CSV `myfile.csv`
  4. Set sheet size to X by Y
  5. Click "Preview SquatchCut"
- What you expected to happen.
- What actually happened (including any error messages).
- Attachments (if possible):
  - The CSV file used
  - Screenshots of the layout or error
  - FreeCAD version
  - SquatchCut version
  - Whether you used metric or imperial units, and CSV units

## Quick links from SquatchCut

In the SquatchCut Task Panel inside FreeCAD, there is a “Report a Bug” button that will open the issues page in your browser.

Remember: more detail = easier to fix.
