# SquatchCut

SquatchCut is a FreeCAD add-on for optimizing rectangular nesting (woodworking, sheet goods).

## Documentation
- **User Guide:** `docs/user_guide.md`
- **Project Guide:** `docs/Project_Guide_v3.3.md` (Architecture & Standards)
- **AI Worker Protocol:** `AGENTS.md` (For AI agents)

## Quick Start
1.  Clone into your FreeCAD Mod directory.
2.  Open FreeCAD.
3.  Select "SquatchCut" workbench.
